# Changelog

## [2.3.0] - 2026-02-08

### Features
- **LLM 智能绘图**: `/draw` 命令支持自动获取并润色 LLM 回复作为提示词
- **Reply 消息支持**: 优先获取 Reply 消息中的 LLM 回复内容
- **中文润色**: 润色后的提示词直接输出中文，更适合 ComfyUI 中文模型
- **指令重命名**: 指令改为 `draw` / `draw_no`，更简洁
- **发送模式**: `draw` 默认直接发送，`draw_no` 为转发模式

### Fixes
- 修复 Telegram event 属性访问错误
- 修复对话历史解析错误
- 修复 `user_id` 非数字时的转换错误

### Improvements
- 添加完整日志便于调试
- 改进对话历史查找逻辑
- 添加更多 ComfyUI API 调试日志
